# lip show

## Usage

```shell
lip show [options] <tooth path>
```

## 功能

展示tooth的信息 如果没有安装，只显示版本列表。

输出是符合RFC标准的邮件头格式。

## 选项

- `-h, --help`

  展示帮助。

- `--files`

  显示已安装文件的完整列表。

- `--available`

  Show the full list of available versions.

- `--json`
  
  Output in JSON format. (cannot be hidden with `--quiet`)
