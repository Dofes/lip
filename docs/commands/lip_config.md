# lip config

## Usage

```shell
lip config [options]
lip config [<key> [<value>]]
```

## Description

Manage configuration.

- If no arguments are specified, list all configuration.
- If a key is specified, print the value of the key.
- If a key and a value are specified, set the value of the key.

## Options

- `-h, --help`

  Show help.
